# pod 使用
第一步 初始化 `pod init`
第二步 加入相关项目
第三步 `pod install`

升级 pod
` gem install cocoapods`

更换repo源
`por repo remove master`

`pod repo add master https://git.coding.net/hging/Specs.git`

`pod repo update`

`pod setup`
